# 项目简介

## 核心功能
- 提供基于YOLOv8的工程机械检测系统，支持对挖掘机、自卸卡车、轮式装载机三类工程机械进行识别
- 支持图片、视频及摄像头三种输入方式进行检测
- 提供完整源码、标注数据集（2655张图片）、训练好的模型及训练结果
- 包含项目运行教程，使用Pycharm+Anaconda或VSCode+Anaconda开发环境
- 界面功能包括：实时显示目标位置、目标总数、置信度；支持批量检测；支持检测结果保存
- 可自定义置信度、IOU阈值；可选择显示标签和原图；可选择检测模型

## 适用场景
- 工程机械种类识别与计数
- 工程施工现场设备监控
- 工程机械相关研究与开发
- 计算机视觉学习与实践

## 技术特点
- 采用YOLOv8目标检测算法
- 使用PyQt5构建用户界面
- 支持Python 3.9环境，依赖opencv-python、ultralytics、torch1.9等库
- 模型已训练完成，可直接使用
- 提供批量处理能力，提高检测效率
